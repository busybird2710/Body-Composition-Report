✅ VERIFIED FORMULAS:

BMI = weight(kg) / [height(m)]²
   Source: WHO Standard

BMR (Male) = (10×weight) + (6.25×height) - (5×age) + 5
BMR (Female) = (10×weight) + (6.25×height) - (5×age) - 161
   Source: Mifflin-St Jeor Equation (±10% accuracy)

Biological Age = Calculated from all biomarker deviations
   Source: Research-based aging algorithm

Wellness Index = 0-100 score based on target range deviations
   Source: Custom wellness scoring system

❓ TROUBLESHOOTING:

Problem: Bluetooth won't connect
Solution: Use Chrome/Edge browser, enable Bluetooth, 
          or use Demo Mode

Problem: Calculations not working
Solution: Fill all required fields, check JavaScript console (F12)

Problem: Report not generating
Solution: Use localhost or deployed version (not file://)

Problem: PDF not exporting
Solution: Use Ctrl+P (print) and "Save as PDF"

Full troubleshooting: IMPLEMENTATION-GUIDE.md, Section 6
