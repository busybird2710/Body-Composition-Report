🔬 HOW TO MANUALLY VERIFY ACCURACY
================================================================================

1. BMI VERIFICATION
   Calculate manually: weight / (height in meters)²
   Example: 70 / (1.75)² = 70 / 3.0625 = 22.86
   Should match app exactly ✓

2. BMR VERIFICATION
   Use Mifflin-St Jeor formula:
   Male: (10 × 70) + (6.25 × 175) - (5 × 30) + 5
       = 700 + 1093.75 - 150 + 5 = 1648.75 ≈ 1649
   Should match app ±1 kcal ✓

3. BODY FAT VERIFICATION
   Use calculator at: calculator.net/body-fat-calculator.html
   Enter BMI and age
   Should be within ±2% of our value ✓

4. BIOLOGICAL AGE VERIFICATION
   Logic check:
   - Overweight → older biological age ✓
   - Low muscle → older biological age ✓
   - High visceral fat → older biological age ✓
   - Fit person → younger biological age ✓
