================================================================================
🔧 HOW IT WORKS
================================================================================

Section-Based Approach:
1. Report is divided into 5 main sections:
   - Header (logo + title)
   - Person Details
   - Parameters Summary
   - Biological Age Summary
   - Footer

2. Each section is captured separately as an image

3. Smart placement algorithm:
   - Check if section fits on current page
   - If yes → Add to current page
   - If no → Start new page
   - If section too tall → Split intelligently

4. Result: Clean page breaks at natural boundaries

================================================================================
📊 PAGE BREAK ALGORITHM
================================================================================

Pseudo-code:

```
For each section:
  1. Capture section as image
  2. Calculate section height
  3. Check available space on current page
  
  If section fits:
    - Add to current page
    - Update current position
  
  Else if section is small:
    - Start new page
    - Add section to new page
  
  Else (section is very tall):
    - Calculate split point
    - Add portion to current page
    - Start new page
    - Add remainder to new page
  
  4. Add small spacing between sections
```

Result: Intelligent placement with no awkward cuts!
