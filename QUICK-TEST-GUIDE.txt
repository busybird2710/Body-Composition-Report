================================================================================
TROUBLESHOOTING
================================================================================

Problem: PDF button does nothing
Solution:
- Check internet connection (needs to load html2pdf.js from CDN)
- Try refreshing the page
- Check browser console (F12) for errors

Problem: PDF downloads but is blank
Solution:
- Wait longer (5-10 seconds) before opening the PDF
- Try using a different browser (Chrome recommended)

Problem: PDF quality is poor
Solution:
- This shouldn't happen (set to 98% quality, 2x scale)
- Try downloading again
- Check if you're using the latest browser version

Problem: Can't find the downloaded file
Solution:
- Check Downloads folder: C:\Users\busyb\Downloads
- Click the download icon in browser (usually top-right)
- Click "Show in folder" next to the PDF file

Problem: Filename is wrong
Solution:
- Make sure you entered a name in the "Person Name" field
- If name has special characters, they may be removed
- Date comes from system time

Problem: Report doesn't generate
Solution:
- Make sure all required fields are filled
- BMI and BMR must be calculated (auto-filled when you enter height/weight)
- Try clicking Auto-Calculate button first
